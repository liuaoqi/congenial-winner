Floating Boxes Demo

Creating the FloatingBoxes instance

/* Create a FloatingBoxGenerator instance created with setId=0 */
const boxes = new FloatingBoxGenerator(0);

Note: FloatingBoxes generated with the given instance will produce div elements having class 'floating-box-0' with the given Id 0.
For the purpose of this demo, boxes will be generated on the top left corner of the webpage that is set in the style sheet, as following:

/* Stylesheet */
.floating-box-0 { left: 10px; }

Creating a FloatingBox

boxes.createBox(id);

Adding DOM Elements to the box

/* Get name of the box using getBoxNameById(id) */
const boxName = boxes.getBoxNameById(i);
const nameElement = document.createElement('h5');
nameElement.innerText = 'Box Name: ' + boxName;
nameElement.style = 'margin: 15px; z-index: 0; overflow: auto;'
/* Adding the DOM element to the box using addDOMById(element, id) */
boxes.addDOMById(nameElement, i);

...

/* Get the box DOM Element using getBoxDOMById(id) */
const box = boxes.getBoxDOMById(i);
const idElement = document.createElement('h5');
idElement.innerText = 'DOM Id: ' + box.id;
idElement.style = 'margin: 15px; z-index: 0; overflow: auto;'
boxes.addDOMById(idElement, i);

Dragging

Enable dragging for all added boxes

boxes.enableDragging();
Click To Make All Added Boxes Draggable

Disable dragging of all added boxes

boxes.disableDragging();
Click To Make All Added Boxes Non-Draggable

Make specific component to be available for dragging

/* The DOM elements will be made draggable by enableDragging(element)
must have Id in the following format:
'floating-box-' + element + sid + '-' + bid */
boxes.enableDragging('border-top');
